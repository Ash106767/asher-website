Cloud Chamber

  • thermal conductivity
  • tolerancing
  • circuits

A Wilson Cloud Chamber that makes the paths of radioactive decay visible.

Build notes

This project was inspired by the design of various modern military equipment, and the noninformative UI design of Star Wars technology. The most challenging part of this project was surprisingly not the chamber itself, but dealing with the warping that appears in prints this large. The cooling method used is a dual peltier plate mounted to a cpu heatsink.

SO-ARM100 Build

  • open source DIY
  • electronics

A pair of 6-axis robotic arms. One arm mimics the position of the other, allowing for fluid remote control of the follower arm.

Build notes

This project was, and is still out of my comfort zone. I couldn't tell you how the code works, but I could tell you that the parts should be printed at a low infill because the tolerancing is incredibly tight for the servos. Had I printed everything at 103% scale, it would have taken half as long. The feeling of interacting with an arm that your other arm is in control of is trippy.
